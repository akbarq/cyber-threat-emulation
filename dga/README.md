
## Description
Emulates DGA based malware traffic.

The script generates random 12 character domains for the following tlds:

`.cc, .top, .ws, .to, .so, .gq, .gdn, .cf`


## Requirements
Python3

## Libraries
Requests.

`pip3 install requests`


## Usage
`chmod +x dga.py`

`./dga.py 5`
